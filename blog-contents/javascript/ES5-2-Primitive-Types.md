# Primitive Type

### 1. String<br>
    "hello"

### 2. Number<br>
    1 / 1.4 / 2934

### 3. Boolean<br>
    true / false

### 4. Null<br>
    null (null값, 값이 없음)

### 5. Undefined<br>
    undefined (정의되지 않음)

### 6. NaN<br>
    Not a Number
    ```javascript
    Math.pow(11, "hello")
    NaN
    ```

    ```javascript
    "hello" / 6
    NaN
    ```

### 7. TypeOf<br>
    ```javascript
    typeof true
    "boolean"
    ```
